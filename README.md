# Programação de IA usando Python

Este repositório contém a implementação de um dos algoritmos de IA, baseados no livro de IA: [Artificial Intelligence: A Modern Approach](https://www.cin.ufpe.br/~gtsa/Periodo/PDF/4P/SI.pdf)

### Primeira parte:

- Conteúdo:
    - Estratégias de busca informadas:
        - Busca A*: minimização do custo total estimado da solução
    - Formulação de problema:
        - Dado um grafo de espaço de estados, o objetivo é procurar um caminho do estado inicial para o estado objetivo.
    - Exemplo:
        - Cidades de Mato-Grosso
